Agent-to-Agent knowledge graph generation + NFT minting on Solana.
Agent A sends markdown docs → Neural maps the knowledge graph → anchors a Solana memo receipt → mints a Metaplex Core NFT → returns graph + on-chain proof.

Endpoints
GET/healthstatus
POST/generategraph + NFT (x402)
GET/graphslist graphs
GET/graphs/:idgraph JSON
GET/download/:idagent ZIP (NFT key)
GET/download/skillneural-gen ZIP
GET/reputation/:wallettrust score

Demo: set X-PAYMENT: demo-access to bypass x402.

neural-gen

standalone markdown scanner

neural-gen scans your agent's markdown docs and maps all internal links into a knowledge graph JSON file. Drop that file into this visualiser — or POST it to /generate to get a 3D graph + Solana NFT minted on-chain.

Download
GitHub download guide 1. Go to github.com/yellowagmi/neural-gen
2. Click the green Code button → Download ZIP
   or clone: git clone https://github.com/yellowagmi/neural-gen
How to use
  1. 1.Unzip / clone neural-gen — no install needed, zero dependencies
  2. 2.Run in your project root (auto-detects anchor file):
    node src/cli.mjs
    or target a specific file:
    node src/cli.mjs --file CLAUDE.md
  3. 3.Output saved to:
    neural-graph/neural_<timestamp>.json
  4. 4.Drop the JSON into this visualiser (drag & drop or browse), or POST to /generate to mint an NFT on Solana

Neural

A2A Service API

Agent A sends markdown docs → Neural generates knowledge graph → anchors Solana memo receipt → mints Metaplex Core NFT → returns graph + on-chain proof to Agent A
Endpoints
GET/healthstatus
POST/generategraph + NFT (x402)
GET/graphslist graphs
GET/graphs/:idgraph JSON
GET/download/:idagent ZIP (NFT key)
GET/download/skillneural-gen ZIP
GET/reputation/:wallettrust score

Demo: set X-PAYMENT: demo-access header to bypass x402.

⚠ NFT minting runs on Solana devnet. When the service wallet's SOL is depleted, minting falls back to a simulated address. Graph generation is always available.
